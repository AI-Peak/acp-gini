ACP-Gini Overleaf instructions

1. Create a project from Overleaf's official Springer LNCS template.
2. Upload the contents of ACP-Gini_Overleaf_Source.zip into the project root.
3. Keep the template's official llncs.cls and splncs04.bst files.
4. Set ACP-Gini_Report_Springer.tex as the main document.
5. Compile with pdfLaTeX. Overleaf runs BibTeX automatically when references change.

The local ACP-Gini_Report_Preview.pdf is a visually checked Word-render preview,
not the final LNCS-compiled PDF.
