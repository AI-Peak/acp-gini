ACP-Gini Overleaf instructions

1. In Overleaf, choose New Project -> Upload Project.
2. Upload ACP-Gini_Overleaf_Source.zip directly; no external template is required.
3. Set ACP-Gini_Report_Springer.tex as the main document if Overleaf does not detect it automatically.
4. Select pdfLaTeX as the compiler. Overleaf runs BibTeX automatically when references change.

The archive is self-contained and includes sn-jnl.cls, sn-basic.bst,
references.bib, generated tables, and all figure files.
